# ZZA
Zerotier auto install script on docker image powered by ztnet app

# Usage
```
sudo su
```
```
git clone https://github.com/safrinnetwork/ZZA
```
```
cd ZZA
```
```
chmod +x install.sh
```
```
sudo apt-get install dos2unix
```
```
dos2unix install.sh
```
```
bash install.sh
```
```
docker compose up -d
```
